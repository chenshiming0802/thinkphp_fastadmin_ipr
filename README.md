CSM 知识产权管理系统赋能企业对企业的商标、专利和著作权的沉淀和管理，替代传统Excel方式。

* 安装 Nginx+Mysql+PHP 运行环境
* 下载源码，网站目录设置指向到 %本项目%/public 目录,以宝塔为例,设置如下
